basically ssh but for people who dont wanna type 50 commands  
you put in an ip + login and it grabs cpu/ram/disk/uptime nerd stuff

---

## what it does

- cli version (terminal)
- web version (localhost:80085)
- shows:
  - hostname
  - uptime
  - ram usage
  - disk usage
  - loadavg cpu stuff
  - os info

---

## install

```bash
pip install -r requirements.txt
```
